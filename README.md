# Image-Watermark-Remover
一款轻量级、跨平台的图片去水印工具，支持**命令行模式**和**图形界面模式**，基于OpenCV图像修复算法实现，无需复杂配置，一键去除文字/简单水印。

## 功能特点
✅ 支持PNG、JPG、JPEG、BMP等主流图片格式；<br>
✅ 双模式：命令行（高效批量处理）+ GUI（可视化操作）；<br>
✅ 自动水印检测，无需手动制作掩码（支持手动掩码优化）；<br>
✅ 轻量级：核心依赖仅OpenCV，跨平台运行（Windows/Mac/Linux）。
